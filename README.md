# Personal Portfolio Website

## Overview
This project is a responsive personal portfolio website designed to showcase my skills, projects, and growth as a web developer. It demonstrates continuous improvement through feature enhancements, performance optimization, and version control using Git.

## Live Demo
https://dhilranv-afk.github.io/my-portfolio/

## Repository
https://github.com/dhilranv-afk/my-portfolio

## Features
- Smooth scrolling navigation
- Dark mode toggle with saved preference (localStorage)
- Project filtering by category (JavaScript-based)
- Responsive design for mobile, tablet, and desktop
- UI animations and hover effects

## Technologies Used
- HTML5 (semantic structure)
- CSS3 (Flexbox, media queries, animations)
- JavaScript (DOM manipulation, event handling)

## Performance Optimization
- Compressed images for faster load times
- Lazy loading implemented using `loading="lazy"`
- Minified CSS and JavaScript
- Removed unused code for efficiency

## Version Control
This project follows a structured Git workflow:
- Multiple commits showing incremental progress
- Descriptive commit messages
- Continuous updates and improvements
