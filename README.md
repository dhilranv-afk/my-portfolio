# Ranveer Dhillon Portfolio

Welcome to my **personal portfolio website**, built to showcase my skills, projects, and learning journey in web development.

This is a beginner-friendly project but styled to look professional and modern. It uses **HTML, CSS**, and basic **animations**, and is fully **responsive** for mobile devices.

---

## 🌟 Features

* **Gradient Hero Section** with a call-to-action button
* **Sticky Navigation Bar** that stays visible while scrolling
* **Smooth Scrolling** for navigation links
* **Animated Project Cards**
* **Google Fonts** for a professional look
* **Mobile Responsive Layout**
* Clean **HTML & CSS structure**
* Ready for **Git version control** and **GitHub Pages deployment**

---

## 🚀 How to Run Locally

1. Clone this repository:

```bash
git clone https://github.com/YOUR-USERNAME/my-portfolio.git
```

2. Navigate to the project folder:

```bash
cd my-portfolio
```

3. Open `index.html` in your browser.

---

## 📱 Mobile Responsiveness

The layout adapts to mobile devices using a media query. Project cards stack vertically, navigation is simplified, and content remains readable.

---

## ⚡ Deployment

This portfolio can be deployed quickly on **GitHub Pages**:

1. Push the repository to GitHub.
2. Go to **Settings → Pages**.
3. Select the **main branch** and `/root` folder.
4. Save – your live portfolio link will be available like:

```
https://yourusername.github.io/my-portfolio
```
