# Personal Portfolio Website

## Overview

This project is a **personal portfolio website** built using **HTML, CSS, and JavaScript**.
The goal of this project is to demonstrate fundamental front-end development skills while creating a professional online presence to showcase projects and contact information.

The portfolio includes responsive design, interactive JavaScript features, and a clean user interface suitable for presenting work to instructors, recruiters, or collaborators.

## Features

The portfolio includes several modern web development features:

* Responsive layout that works on desktop and mobile devices
* Gradient hero section with call-to-action button
* Animated typing effect in the hero section
* Featured projects layout
* Interactive project cards
* Project modal popup gallery
* Contact form with basic JavaScript validation
* Smooth scrolling navigation
* Sticky navigation bar
* Modern footer with social media links

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Launch **index.html** in a web browser.

No build tools or installations are required.

## JavaScript Functionality

JavaScript is used in the portfolio to provide interactive features:

* Typing animation in the hero section
* Modal popup displaying project details
* Contact form validation
* Event handling for user interactions

These features demonstrate basic understanding of **DOM manipulation, event listeners, and user interface behavior**.
